 ### Git Clone, Package Install(Terminal)

```bash
git clone https://github.com/godee95/gyeonggi.git
npm i
```

### .env 파일 생성(Terminal)
```bash
touch .env
```

### .env 안 설정값
```
PORT=4000
MYSQL_AWS_PASSWORD=12345678
MYSQL_AWS_HOST=AWS_RDS_EndPoint
SECRET_KEY=My_SecretKey
```

 ### 실행(Terminal)

```bash
node app.js
```
